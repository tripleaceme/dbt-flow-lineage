# Changelog

## [0.1.0] - 2026-03-21

### Added
- Column-level lineage visualization with animated particle flow
- Four transformation types: passthrough (blue), rename (green), transform (yellow), aggregate (purple)
- Model-focused view: right-click `.sql` file to see only that model's lineage
- Upstream/downstream direction filter
- Draggable model nodes with automatic edge redrawing
- Zoom controls: `+`/`-` keyboard shortcuts and UI buttons
- Column-level dimming: unlinked columns grey out when a column is selected
- Activity Bar sidebar with model/column tree
- CodeLens annotations above SELECT statements
- Colored terminal startup banner
- Welcome page on first install
- Auto-refresh when manifest.json changes
- Multi-project workspace support with project picker
- Persistent project selection across VS Code relaunches
